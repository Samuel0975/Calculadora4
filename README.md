# Calculadora
Diseñe una calculadora que hace las funciones basicas de una calculadora tiene la funcion de calcular cualquier tipo de numeros ya sea fraccinario etc.

para hacer esta calculadora utilice textBos, font y Button, tex, name.

Button son los botones, que lleva cada valor en la calculadora, font se utiliza para hacer mas grandes o pequeños los numero y signos de la calculadora, y el textBox lo utilice para crear la pantalla, donde se colocan los numeros.
